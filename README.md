# README — Pasta de imagens do exercício Mais Você

## Finalidade da pasta

Esta pasta contém **somente os arquivos de imagem** utilizados no exercício da landing page do **Mais Você**.

Ela serve como apoio visual para a construção da página em HTML e CSS.

---

## O que há nesta pasta

Nesta pasta devem ficar apenas imagens, como por exemplo:

* logo do programa
* imagem horizontal para a parte superior da página
* imagem vertical para a lateral do formulário


---

## Como usar as imagens no projeto

As imagens desta pasta devem ser chamadas no HTML com o caminho correto.

Exemplo:

```html
<img src="imagens/logo.webp" alt="Logo do programa">
```

Se a pasta se chamar `imagens`, o caminho deverá começar com `imagens/`.

---

---

## Função pedagógica das imagens

As imagens desta pasta ajudam o aluno a praticar:

* inserção de imagens no HTML
* organização visual da página
* distribuição de elementos com CSS
* composição de uma landing page mais próxima de um caso real

---

## Estrutura esperada

Exemplo:

```text
/projeto
  index.html
  style.css
  script.js
  /imagens
    logo.webp
    foto-topo.jpg
    foto-lado.jpg
```

---

## Objetivo no exercício

O aluno deverá utilizar as imagens desta pasta para montar uma landing page com:

* logo na parte superior
* imagem horizontal no topo
* imagem vertical ao lado do formulário
* melhor composição visual da página

---

## Verificação final

Antes de concluir, confira:

* se todas as imagens abrem normalmente
* se os nomes estão corretos
* se o HTML aponta para o caminho certo
* se nenhuma imagem está quebrada na página

---

## Observação

Esta pasta é apenas de apoio visual.
Ela não contém lógica, estilos ou estrutura da página.
Seu papel é fornecer os arquivos necessários para montar o layout do exercício.
